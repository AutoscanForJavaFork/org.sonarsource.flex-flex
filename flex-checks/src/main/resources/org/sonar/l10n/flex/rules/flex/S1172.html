Unused parameters are misleading. Whatever the value passed to such parameters is, the behavior will be the same.

Noncompliant Code Example

function doSomething(a:int, b:int):void      // "b" is unused
{
  compute(a);
}

Compliant Solution

function doSomething(a:int):void      // "b" is unused
{
  compute(a);
}

Exceptions

Methods overriding or implementing another one should be excluded

override function doSomething(a:int):void     // no issue reported on b
{
  compute(a);
}
