Switch cases should remain small to keep the overall switch compact and readable.

Noncompliant Code Example

The following code snippet illustrates this rule with the default threshold of 5:

switch (myVariable) {
  case 0:       // Non-Compliant - 6 lines till next case or default case
    trace("");
    trace("");
    trace("");
    trace("");
    break;
  case 1:
  ...
}

Compliant Solution

switch (myVariable) {
  case 0:
    printSomething()
    break;
  case 1:
  ...
}
...
private void printSomething() {
  trace("");
  trace("");
  trace("");
  trace("");
}
